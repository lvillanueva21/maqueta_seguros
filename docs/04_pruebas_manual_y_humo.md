# Pruebas — BS-20260627-223240-PRINTFIX

1. Inicia sesión como Gerente o Ejecutivo.
2. Abre una cotización existente o crea una con datos mínimos válidos.
3. Pulsa **Vista A4**.
4. Confirma que la hoja se vea dentro de la aplicación.
5. Pulsa **Imprimir / Guardar como PDF**.
6. Verifica que la vista de impresión muestre encabezado, datos del destinatario, grupos y alternativas.
7. Selecciona **Guardar como PDF** y confirma que el archivo generado no esté vacío.
8. Cancela una impresión y vuelve a intentar; la vista debe seguir disponible.
